Origem do MX Linux

O MX Linux é a junção das empresas antix e MEPIS em dezembro de 2013. O nome “MX” foi escolhido para combinar a primeira letra de Mepis com a última de antiX. Para ser listado no DistroWatch, o MX Linux foi apresentado como uma versão do antiX e lançou sua primeira versão em março de 2014.
